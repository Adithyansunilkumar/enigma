# ENIGMA - CSE Association

The official Computer Science & Engineering association website for Nehru College of Engineering and Research Centre (NCERC).

## Features
- **Hero Section**: Encoding Tomorrow branding.
- **About Section**: Information about the ENIGMA association.
- **Events & Workshops**: Showcase of upcoming and past events.
- **Team**: Meet the core team members.
- **Contact**: Reach out to the association.

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Framer Motion** (for animations)
- **Lenis** (for smooth scrolling)
- **Lucide React** (icons)

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `src/sections`: Main sections of the application.
- `src/components`: Reusable UI components.
- `public/assets`: Images and icons.
